# Karna's Arch Dotfiles inside WSL

This repository contains my personal configuration files (dotfiles) for **Arch Linux** inside Windows Subsystem for Linux (WSL).

> [!WARNING]
> **Personal Use Only:** Please do not create Issues or Pull Requests. These dotfiles are tuned for my hardware and username. Some configs may contain **absolute paths**. If something crashes, please check the file paths first.

> [!NOTE]
> **Setup:** Arch Linux in WSL setup is in [Arch Linux WSL Setup Guide](./Arch_WSL2_Setup_Guide.pdf) file. After that you can setup the dotfiles using stow.
